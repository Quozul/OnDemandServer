OnDemandServer API

Welcome to the home page of the OnDemandServer proxy plugin. Here you will find the different endpoints the API has.

Endpoints

  • /
    : this page
  • /status/:?name
     : status of all servers or the given name
  • /start/:name
    : starts the server with the given name
  • /stop/:name
    : stops the server with the given name
  • /user/:uuid
    : get all on-the-fly created servers by a user
